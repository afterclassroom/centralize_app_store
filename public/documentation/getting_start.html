API TERMS OF USE

Before you start using the API, we have a few guidelines that we'd like you to know. Please make sure to read the full API Terms of Use. Here's a quick summary:

1. gotoClassroom users own their media. It's your responsibility to make sure that you respect that right.
2. You cannot use the gotoClassroom name in your application.
3. You cannot use the gotoClassroom API to crawl or store users' media without their expressed consent.
4. You cannot replicate the core user experience of www.gotoclassroom.com/api
5. Do not abuse the API. Too many requests too quickly will deny you access.
